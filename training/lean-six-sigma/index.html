Program Focus

Lean Six Sigma Yellow Belt / Foundation Level


Designed for learners who want a structured introduction to Lean Six Sigma tools, improvement thinking, and project-based application.

What You Will Learn

This training focuses on practical tools that can be applied in academic projects, workplace improvement, operations, quality, and analytics-related roles.

Lean Thinking

Identify waste, non-value adding activities, bottlenecks, and improvement opportunities in real processes.

DMAIC Methodology

Understand the Define, Measure, Analyze, Improve, and Control cycle as a structured problem-solving framework.

Quality Tools

Apply basic tools such as SIPOC, process maps, 5 Whys, fishbone diagram, Pareto analysis, and control planning.

Program Coverage

The course is organized into beginner-friendly modules with emphasis on application, not just theory.

Module 1: Introduction to Lean Six Sigma and continuous improvement
Module 2: Lean principles, waste identification, and value-added analysis
Module 3: DMAIC framework and project improvement structure
Module 4: Problem statement, goal statement, SIPOC, and process mapping
Module 5: Root cause analysis using 5 Whys, fishbone diagram, and Pareto analysis
Module 6: Measurement basics, performance indicators, and data-driven decision making
Module 7: Improve and Control phase tools, sustainment planning, and certification preparation
